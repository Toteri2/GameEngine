# Welcome to R-Type Game Engine Documentation

Welcome to the **R-Type Game Engine** documentation! This project is a modern C++ implementation of the classic R-Type horizontal shoot'em up game, featuring a custom-built game engine with networking capabilities.

## What is R-Type?

R-Type is a classic horizontal scrolling shooter arcade game. This project recreates the legendary gameplay experience using modern C++ practices and software engineering principles, while implementing a full-featured game engine from scratch.

## Key Features

- **Entity-Component-System (ECS) Architecture**: Modular and performant game engine design
- **Network Multiplayer**: Client-server architecture supporting multiple players
- **Binary Protocol**: Custom UDP-based network protocol for real-time gameplay
- **Cross-Platform**: Built with CMake and designed to run on Linux and Windows
- **Modern C++**: Leveraging C++23 features and best practices

## Project Structure

The project is organized into three main components:

- **Engine**: The core ECS game engine with modular systems
- **Server**: Multi-threaded game server handling game logic and state
- **Client**: Graphical client for rendering and player input
